# QuickBoard（Frontend）

## 端口约定
- 前端（Vite）：`http://127.0.0.1:5173`
- 后端（API + Socket）：`http://127.0.0.1:3000`

前端通过环境变量 `VITE_API_URL` 指向后端（默认建议 `http://127.0.0.1:3000`），白板的公式识别等能力由后端再转发到 OCR 服务。

## 开发运行
```sh
npm install
npm run dev -- --host 127.0.0.1 --port 5173 --strictPort
```

## 使用方式
- 访问首页 `http://127.0.0.1:5173/`：创建/加入房间，并显示最近房间。
- 白板页 `http://127.0.0.1:5173/board?room=demo-room`：直接进入指定房间。
- 兼容旧链接 `http://127.0.0.1:5173/?room=demo-room`：会自动跳转到白板页。

## 构建
```sh
npm run build
```

## 单测与规范
```sh
npm run test:unit
npm run test:unit -- --run
npm run lint
```

## 弱网模拟（开发验证）
在 URL 上添加参数即可开启（用于复现延迟/抖动/丢包）：

```
http://127.0.0.1:5173/board?room=demo-room&netsim=1&drop=0.1&delay=200&jitter=200
```

## UI/UX 小技巧
- 工具快捷键：`V` 选择、`P` 画笔、`E` 橡皮、`R` 矩形、`C` 圆形。
- 形状创建：切到 `R/C` 后，在画布上 **按住左键拖拽** 创建；`Esc` 可取消。
- 工具环：在白板区域 **鼠标右键按住**（或触屏 **长按**）呼出半透明“扇形工具环（调色盘）”，拖到扇形图标上松开即可切换工具。
- 视图：按住 `Space` 拖拽平移；`Ctrl/⌘ + 0` 复位；`Ctrl/⌘ + +/-` 缩放；双击画布复位视图。
- 帮助：按 `?` 或点工具栏“帮助”。

## 视觉与微动效
- 进入页与全局背景：轻网格 + 轻噪点 + 光晕渐变（不影响白板导出）。
- 进入页操作反馈：创建/加入房间时会有轻量“火花”粒子提示。
- 远端光标：按用户稳定配色，并带轻量拖尾与发光，方便区分成员。
